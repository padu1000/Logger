This is a small bash script for logging your thoughts.
It is simple to use, just change the permissions so it works on your
local machine.

Works with: Mac, Linux, and any other bash distribution

To exit press Control^C

The program will make a new file once started.  Press return twice in order
to log a comment.  Each comment is date and time stamped.
